Templates are pre-built strategies with triggers and rules already configured. Clone one, tweak if you like, and activate. No need to design from scratch.

### When to use a template

- **You're new** — Get value fast. Fort Knox (Max Safety) and similar templates filter scams and low-effort launches so you can learn what good alerts look like.
- **You want a baseline** — Clone, then adjust triggers and rules to match your risk tolerance.
- **You're exploring** — Try different styles (conservative, balanced, aggressive) before building your own.

### How to use a template

1. Click **+ Create New Strategy** on the Dashboard or Strategies page.
2. Choose **Start from Template**.
3. Browse the list. Each template shows name, description, and often metrics (win rate, winners) if available.
4. Select one and click **Use Template**.
5. You'll enter the Strategy Editor with triggers and rules pre-filled. Rename the strategy, adjust anything you want, then save and activate.

<Tip>
You can clone multiple templates and run them in parallel. Each becomes its own strategy. Compare Performance across them to see which style works best for you.
</Tip>
