---
title: "Full Trigger Glossary"
description: "Events that cause strategies to run—the 'when' of your alerts."
---

* **A New Token is Created**
  * Description: Fires when a token is added on the solana chain or on meme platforms like pump.fun

* **A Tweet Mentions a Token **
  * Description: Fires when a tweet matching your settings and author list mentions a token (cashtag or contract)

* **A Narrative Emerges (Pro)**
  * Description: Fires when a new token matches an AI‑identified narrative from your monitored tweet handles

* **A Followed Dev Deploys a New Token **
  * Description: Fires when a developer from My Developers creates a new contract token

* **A Token Matches a Search token filter**
  * Description: Fires when your scheduled search token filter finds a token matching all filter criteria
