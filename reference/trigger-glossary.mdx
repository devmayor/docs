- **A New Token is Created**
  - Description: Fires when a token is added on the Solana chain or on meme platforms like pump.fun

- **Almost Graduated Tokens**
  - Description: Fires when tokens have gone past the newly created stage and are heading for graduation

- **Graduated Tokens**
  - Description: Fires when a token has finally graduated from platforms like pump.fun

- **A Tweet Mentions a Token**
  - Description: Fires when a tweet mentions a token either by contract address (CA) or token name. Matches your settings and author list.

- **AI Generated Tokens from Tweets**
  - Description: Fires when possible token names are generated from tweets and are found to be newly created. For example, if Elon tweets about "colony on mars," the AI will predict and create likely token names from his tweets like `Mars` or `Mars doge`. When new launches are created with this likeness, they will trigger your rules to evaluate.

- **A Followed Dev Deploys a New Token**
  - Description: Fires when a developer you follow launches a new token
